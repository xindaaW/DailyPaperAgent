# Baseline Comparator Skill

## Purpose
对每条主题中的新论文做“同主题 baseline 比较”，避免机械套用统一基线。

## Use this skill when
- 需要解释创新点是否真实成立。

## Core rules
- 只能与“主题相近”的方法比较，不得跨赛道乱比。
- 比较维度至少包含：方法假设、训练信号、数据依赖、推理成本、效果边界。
- 不得只写“优于 baseline”，必须说明为何优/何时不优。

## Standard workflow
1. 识别论文所属主题。
2. 检索该主题近作与代表 baseline。
3. 形成差异表：新增机制、收益来源、失败条件。
4. 给出可信结论和风险项。
