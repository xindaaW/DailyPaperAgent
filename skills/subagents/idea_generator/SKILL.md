# Idea Generator Skill

## Purpose
基于当天论文与历史脉络，提出可落地的后续研究 idea。

## Use this skill when
- 需要从“读论文”转到“可执行研究计划”。

## Core rules
- 每个 idea 必须包含：问题定义、核心假设、实验设计、评估指标、失败判据。
- 优先提出小步快跑可验证方案。
- 不得给空泛方向词（如“提升泛化能力”）而无实验路径。

## Standard workflow
1. 读取洞察与未解决问题。
2. 生成 3-5 个候选 idea。
3. 过滤不可验证/代价过高方案。
4. 输出可执行 idea 卡片。
