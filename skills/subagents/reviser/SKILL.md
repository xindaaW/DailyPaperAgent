# Reviser Skill

## Purpose
根据 reviewer 或质量闸门反馈，对日报草稿做定向增强与修订。

## Use this skill when
- 已有草稿和明确批评意见，需要快速提升完整性、证据链和可读性。

## Core rules
- 必须逐条响应 critique 中的关键问题，不能只做表面润色。
- 优先补证据和结构，再优化措辞；不能为了流畅删掉重要论文信息。
- 保留论文标题、链接、baseline 对比、洞察来源和 idea 的实验路径。
- 改写后应让文章更像完整报告，而不是“原文 + 几条补丁”。

## Standard workflow
1. 解析 critique，识别必须修复的问题。
2. 对照现有草稿与论文上下文，补齐缺失信息和证据链。
3. 重组段落顺序，消除碎片化表达和重复表述。
4. 输出修订后的完整中文稿，确保关键反馈都被处理。
