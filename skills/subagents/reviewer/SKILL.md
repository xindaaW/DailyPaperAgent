# Reviewer Skill

## Purpose
对当前日报草稿进行严格审阅，指出会影响发布质量的具体问题。

## Use this skill when
- 已经有初稿，但还不确定是否覆盖完整、证据是否扎实、表达是否可发布。

## Core rules
- 优先找高影响问题：缺论文链接、baseline 对比失真、洞察无证据、idea 不可执行、结构松散。
- 批评必须具体到问题片段或问题类型，不能只说“内容不够好”。
- 每条问题都要附带可执行修改建议，方便后续 reviser 直接落地。
- 审稿目标是提升终稿质量，不是重写全文。

## Standard workflow
1. 检查四类核心内容是否齐全：论文更新、baseline 差异、领域启发、后续 idea。
2. 检查证据链是否闭合：结论是否能追溯到具体论文或比较结果。
3. 检查可读性：是否短句堆砌、段落失衡、主线不清。
4. 输出“问题清单 + 修改建议”，按严重程度排序。
